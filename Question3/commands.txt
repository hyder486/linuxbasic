
### Command : echo "This is sample data for link testing." > ~/sample_data.txt
Creates sample_data.txt in home directory and writes sample text into it.

### Command : ln ~/sample_data.txt ~/sample_hard.txt
Creates a hard link named sample_hard.txt pointing to the same inode as sample_data.txt.

### Command : ln -s ~/sample_data.txt ~/sample_soft.txt
Creates a symbolic link that stores the path to sample_data.txt, not the inode.

### Command : ls -li ~/sample_data.txt ~/sample_hard.txt ~/sample_soft.txt
shows inode numbers and detailed listing

### Command : stat ~/sample_data.txt
Displays permissions, owner, group, size, timestamps, inode number, and link count.

### Command : -sh ~
Shows total disk space used by your home directory.

### Command : ls -lh ~
Lists files in your home directory with human-readable sizes.
