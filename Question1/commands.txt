### Command : id
displays curently logged in username, UID, GID and all groups the account belongs to

### Command : pwd
shows the current working directory

### Command : ls -l
lists all files and directories in long format

### Command : echo "Linux user environment verified" > user_info.txt
Creates a file named user_info.txt and writes the required line into it.

### Command : wc -m user_info.txt
Displays the number of characters present in user_info.txt.

### Command : man mkdir
Access the manual page of the mkdir command

### Command : mkdir -p dir1/dir2
The -p option creates parent directories automatically and prevents errors if directories already exist.

### Command : ls ~
Lists the contents of your home directory, sorted alphabetically by default.

### Command : grep "admin" log.txt
Searches for the word "admin" in log.txt and displays only the matching lines.

### Command : uname -r
Displays the currently running Linux kernel version.

### Command : ping -c 4 www.google.com
Sends 4 ICMP packets to verify network connectivity.
Successful replies indicate the system has internet access.

### Command : uptime
displays how long the system has been running, the number of currently logged in users and the CPU load average
