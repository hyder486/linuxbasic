### Command : mkdir ~/documents
Creates a directory named documents inside the home directory.

### Command : cd ~/documents
cd moves into the documents directory

### Command : touch plan.txt
touch creates an empty file named plan.txt

### Command : echo "Initial project plan: define requirements and timelines." > plan.txt
Writes a sample project note into plan.txt.

### Command : ls -l plan.txt
Displays permissions, owner, group, size, and timestamps.
Username appears as the file owner.

### Command : cp plan.txt plan_copy.txt
Creates a duplicate of plan.txt named plan_copy.txt.
